* tasseo-py

** Intro
   
Python implementation of [[https://github.com/obfuscurity/tasseo][tasseo]].

** Deployment

#+BEGIN_SRC shell
  $ pip install -r requirements.txt
  $ GRAPHITE_URL=http://... python code.py
#+END_SRC

** Configuration

Please ref [[https://github.com/obfuscurity/tasseo][tasseo]].
